# 🤖 אוטומציה מלאה ליצירת תמונות AI

## מה זה עושה?

סקריפט זה פותח את כלי יצירת התמונות AI (Gemini/DALL-E/Leonardo) בדפדפן אוטומטי, מדביק את ה-prompts, ומוריד את התמונות - **הכל אוטומטית!**

## 🎨 כלים נתמכים

- **Gemini** (Google) - חינמי! ✅ מומלץ
- **ChatGPT/DALL-E** (OpenAI) - בתשלום
- **Leonardo AI** - חינמי עם מגבלות
- **Ideogram** - חינמי עם מגבלות
- כל כלי אחר שפועל בדפדפן

## 🚀 התקנה

```bash
cd automation
npm install puppeteer
```

## ⚙️ הגדרה

### שלב 1: בחר את הכלי

ערוך את `nano-banana-automation.js` (שורה 7):

```javascript
// בחר אחד מהבאים:
this.imageGenUrl = 'https://gemini.google.com/app';  // Gemini - מומלץ!
// this.imageGenUrl = 'https://chatgpt.com';         // ChatGPT/DALL-E
// this.imageGenUrl = 'https://leonardo.ai';         // Leonardo AI  
// this.imageGenUrl = 'https://ideogram.ai';         // Ideogram
```

### שלב 2: הפעל אוטומציה

צור קובץ `.env`:
```bash
AUTO_GENERATE_IMAGES=true
```

### שלב 3: הרץ

```bash
node simple-automation.js
```

הדפדפן ייפתח אוטומטית, יצור את התמונות, וישמור אותן!

## 🎯 איך זה עובד?

1. **פתיחת דפדפן** - Puppeteer פותח Chrome אוטומטי
2. **ניווט לאתר** - מגיע ל-Nano Banana
3. **הדבקת Prompt** - מזהה את שדה ה-prompt ומדביק אוטומטית
4. **לחיצה על Generate** - מוצא ולוחץ על הכפתור
5. **המתנה לתמונה** - ממתין עד שהתמונה מוכנה (1-2 דקות)
6. **הורדה** - שומר את התמונה בתיקייה הנכונה
7. **חזרה על התהליך** - עושה את זה שוב לאינפוגרפיק

## ⏱️ כמה זמן זה לוקח?

- **תמונת Hero**: ~1-2 דקות
- **אינפוגרפיק**: ~1-2 דקות
- **סה"כ**: ~3-4 דקות לבלוג מלא

## 🐛 פתרון בעיות

### הדפדפן לא מוצא את שדה ה-Prompt

הסקריפט ינסה למצוא אוטומטית, אבל אם לא מצליח:
1. הוא יציג את ה-prompt
2. תעתיק ידנית
3. תלחץ Enter להמשיך

### התמונה לא נמצאת אוטומטית

1. הסקריפט יחכה עד 2 דקות
2. אם לא מצליח, יבקש ממך לשמור ידנית
3. שמור בשם `hero.png` או `infographic.png` בתיקייה המוצגת

## 💡 טיפים

- **השאר את הדפדפן גלוי** (headless: false) כדי לראות מה קורה
- **אל תסגור את הדפדפן** בזמן שהתמונה נוצרת
- **המתן בסבלנות** - יצירת תמונות לוקחת זמן

## 🎨 אחרי יצירת התמונות

הרץ את סקריפט האופטימיזציה:

```bash
npm run optimize-blog-images
```

זה יצור:
- גרסאות WebP (קטנות ב-90%)
- 3 גדלים: 1200px, 800px, 400px
- אופטימיזציה למכשירים שונים
