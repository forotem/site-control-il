- Prompt כללי מדי
- לא מספיק פרטים ויזואליים

**אחרי:**

### 2. ✅ Infographic Timeline - כעת עשירה במידע מהבלוג
**לפני:**
- ציר זמן כללי בלבד
- לא מספיק סטטיסטיקות

**אחרי:**
- **שמאל**: מערכת מסורתית (2020)
  - חיישן תנועה בסיסי
  - 90% אזעקות שווא
  - ניטור ידני
  
- **אמצע**: חץ התפתחות (2021→2026)
  - קו עלייה טכנולוגי
  
- **ימין**: מערכת חכמה (2026)
  - AI זיהוי פנים ועצמים
  - <5% אזעקות שווא
  - אוטומציה מלאה
  
- **סטטיסטיקות**:
  - חיסכון בביטוח: 30%
  - חיסכון בחשמל: 25%
  - זמן תגובה: 2 שניות
  - כושר אינטגרציה: 50+ מכשירים

### 3. ✅ Comparison Chart - כעת מפורטת ברורה
**לפני:**
- תיאור כללי

**אחרי:**
- 6 קטגוריות השוואה ברורות:
  1. **זיהוי איומים**: חיישן בסיסי vs. AI neural network
  2. **אזעקות שווא**: 90% vs. <5%
  3. **זמן תגובה**: שעות vs. 2-3 שניות
  4. **אינטגרציה**: עצמאית vs. 50+ מכשירים
  5. **ניטור**: ידני vs. אוטונומי
  6. **למידה**: כללים סטטיים vs. machine learning
  
- צבעים משמעותיים:
  - 🟢 ירוק: יתרונות מערכת חכמה
  - 🔴 אדום: מגבלות מערכת מסורתית

## 🚀 איך להמשיך עכשיו:

2. בחר תמונה (Hero, Timeline, או Comparison)
3. העתק את הprompt המלא מה-JSON
4. כוונן פרטים קטנים אם צריך
5. ייצא בפורמט WebP

### אפשרות 2: Gemini AI
```bash
# כאשר תהיה API key:
export GEMINI_API_KEY=your_key
npm run generate-images
```
